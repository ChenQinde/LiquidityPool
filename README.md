# LiquidityPool: Game-Theoretic Analysis of Stakeholder Revenue in Ranking-Dependent DeFi

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18330360.svg)](https://doi.org/10.5281/zenodo.18330360)

This repository contains the implementation code and experimental materials 
for our WWW 2026 paper.

## Paper Information

**Authors:** Qinde Chen, Huawei Huang, Jian Zheng

**Conference:** The Web Conference 2026 (WWW '26)

**Paper DOI:** [10.1145/3774904.3792260](https://doi.org/10.1145/3774904.3792260)

## Repository Contents

- `src/` - Source code implementation, include a little demo
- `data/` - Data from the transaction from ETH
